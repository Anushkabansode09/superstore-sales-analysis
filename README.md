# Superstore Sales Analysis

EDA on Sample Superstore dataset using Python + Power BI dashboard to uncover sales and profitability insights.

## Tools
Python (Pandas, Matplotlib, Seaborn) · Power BI · Jupyter Notebook

## Dataset
~9,994 orders · 2014–2017 · Categories: Furniture, Office Supplies, Technology

## Key Insights
- Discounts above 30% consistently lead to negative profit margins
- Tables and Bookcases are loss-making despite decent sales volume
- West region highest sales; Central region worst profit margins
- Q4 seasonal spike every year across all categories

## Project Structure

├── data/Sample_Superstore.csv
